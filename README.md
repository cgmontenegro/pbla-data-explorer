# JupyterLab para exploração de dados do PBL.A

JupyterLab com acesso aos dados do PBL Analytics, baseado na imagem do docker [jupyter/scipy-notebook](https://hub.docker.com/r/jupyter/scipy-notebook).

Para rodar esta aplicação, clone o repositório e gere o container a partir do Dockerfile.

*Em desenvolvimento como parte do Mestrado de Engenharia de Software da CESAR School.*
